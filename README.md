
## Project Setup

```sh
npm install
```
### Before running the application set env variables


```shell
# Create .env file
cp .env.example .env
```

### Compile and Hot-Reload for Development




```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
